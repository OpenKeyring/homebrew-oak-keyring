# homebrew-oak-keyring

Homebrew tap for [ok](https://github.com/OpenKeyring/oak-keyring) — a terminal-first password manager.

## Install

```bash
brew tap openkeyring/oak-keyring
brew install ok
```

## Update

```bash
brew upgrade ok
```

## Uninstall

```bash
brew uninstall ok
brew untap openkeyring/oak-keyring
```
